# Laravel and Vue js Todo App

## Run Locally

Clone the project

```bash
  git clone https://github.com/jaysontolentino/laravel-vue-todo.git
```

Go to the project directory

```bash
  cd laravel-vue-todo
```

## Install laravel and run api server locally

Go to the api directory
```bash
  cd api
```

## Requirements
| Requirement | Version   |
|-------------|-----------|
| PHP         |   8.1.10  |

## Installation
Make sure all requirements are installed on the system.
install dependencies:
```bash
$ composer install
```

## Generate App Key
Generate the application key using the following command:
```bash
$ php artisan key:generate
```

## Migration
Run the migrations table:
```bash
$ php artisan migrate
```

## Run laravel Application
Run the server:
```bash
$ php artisan serve
```



## Install VueJS and start server

Go to frontend directory
```bash
  cd ../frontend
```

```bash
  npm install
```

```bash
  npm run dev
```
